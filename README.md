# LeetCode Tracker

A personal LeetCode progress tracking dashboard built with Python, Streamlit, Pandas, and Plotly.

## Features

* Track solved, attempted, and review problems
* Difficulty analytics (Easy, Medium, Hard)
* Topic analytics
* XP and leveling system
* Daily streak tracking
* Progress-over-time visualization
* Weak topic analysis
* CSV data persistence
* Download progress as CSV
* Clickable LeetCode problem links

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* Git
* GitHub

## Running Locally

```bash
pip install streamlit pandas plotly
streamlit run leetcodetracker.py
```
